# Краткая инструкция по использованию системы

## Быстрый старт

### Для Windows (PowerShell)

1. Откройте PowerShell в директории проекта
2. Выполните команду:
   ```powershell
   .\run.ps1
   ```

Скрипт автоматически:
- Создаст виртуальное окружение (если отсутствует)
- Установит зависимости
- Применит миграции
- Предложит загрузить тестовые данные
- Запустит сервер разработки

### Ручная установка

```powershell
# 1. Создание виртуального окружения
python -m venv venv

# 2. Активация
.\venv\Scripts\activate

# 3. Установка зависимостей
pip install -r requirements.txt

# 4. Настройка .env
copy .env.example .env

# 5. Миграции
python manage.py makemigrations
python manage.py migrate

# 6. Тестовые данные (опционально)
python manage.py populate_data

# 7. Создание суперпользователя (опционально)
python manage.py createsuperuser

# 8. Запуск сервера
python manage.py runserver
```

## Первый вход

После запуска проекта и загрузки тестовых данных доступны следующие учётные записи:

### Клиент
- **Логин:** client
- **Пароль:** client123
- **Доступ:** Просмотр и создание своих заявок, получение чеков

### Работник сервисного центра
- **Логин:** worker
- **Пароль:** worker123
- **Доступ:** Управление запасами, обработка заявок, отчёты

### Управляющий сервисного центра
- **Логин:** manager
- **Пароль:** manager123
- **Доступ:** Полный доступ ко всем функциям

## Основные функции

### Для клиента
1. Войдите в систему
2. Перейдите в "Панель управления"
3. Используйте кнопку "Создать заявку"
4. Заполните форму и отправьте

### Для работника
1. Войдите в систему
2. В меню "Склад" управляйте комплектующими
3. В разделе "Заявки" обрабатывайте запросы клиентов
4. Добавляйте позиции в заявки
5. Отмечайте заявки как выполненные

### Для управляющего
1. Полный доступ ко всем разделам
2. Управление пользователями в разделе "Пользователи"
3. Просмотр аналитики в "Отчёты"
4. Контроль всех процессов

## Структура меню

### Навигация
- **Панель управления:** Главный дашборд с актуальной информацией
- **Склад:** Управление комплектующими, категориями, поставщиками
- **Заявки:** Список и управление заявками
- **Отчёты:** Формирование отчётности (для работников и управляющих)
- **Пользователи:** Управление пользователями (только для управляющих)

### Профиль
- Просмотр и редактирование личной информации
- Выход из системы

## Решение проблем

### Сервер не запускается
```powershell
# Проверьте, не занят ли порт 8000
netstat -ano | findstr :8000

# Запустите на другом порту
python manage.py runserver 8080
```

### Ошибки миграций
```powershell
# Удалите db.sqlite3 и начните заново
Remove-Item db.sqlite3
python manage.py migrate
python manage.py populate_data
```

### Забыли пароль администратора
```powershell
# Создайте нового суперпользователя
python manage.py createsuperuser
```

### Ошибки импорта модулей
```powershell
# Переустановите зависимости
pip install -r requirements.txt --force-reinstall
```
